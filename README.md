# DP Core - Phase1 Substrate V3

Production-grade foundational substrate for reflective cognition experimentation.

## Features

- Reflective cognition loop
- Observation → Abstraction → Theory → Validation → Reflection
- Local-first Ollama execution
- PostgreSQL memory foundation
- Neo4j lineage graph foundation
- DuckDB market memory foundation
- Future-ready topology
- Poetry-managed environment

## Setup

### Install dependencies

```bash
poetry install
```

### Ensure Ollama is running

```bash
ollama serve
```

### Verify llama3.2 exists

```bash
ollama list
```

### Run cognition loop

```bash
poetry run python -m bootstrap.run_cognition_loop
```

# DP / EkamNet

## Vision

Reflective intelligence evolving through replay, contradiction, memory, and self-critique.

Prediction should emerge indirectly from epistemically healthy cognition.

## Why Market Domain

1. Real-world structured variability.
2. Fast feedback loop.
3. First validation environment for longitudinal cognition.

Market is a validation domain, not the product.

## Core Layers

Deterministic:

- Replay.
- Contradiction registry.
- Confidence evolution.
- Persistence.
- Observability.

Generative:

- Abstraction.
- Theory mutation.
- Reflection.

Lifecycle:

- Active.
- Dormant.
- Contradicted.
- Retired.
- Revived.

## Replay Lifecycle

Exact order:

```text
observe
retrieve memory
abstract
mutate/new
detect contradictions
retire stale
revive matched
validate
reflect
persist
```
